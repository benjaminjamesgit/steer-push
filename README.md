# steer-push

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20947370.svg)](https://doi.org/10.5281/zenodo.20947370)

Does a physical difference **STEER** a system (selection) or merely **PUSH** it (causation)?
And can that distinction be drawn **below life** — with no chooser, no self-maintaining
beneficiary, and no observer?

This is the empirical arm of a naturalist/physical-semiotics question: when does a physical
difference become a *sign* rather than just a *cause*? The slot is unoccupied — persistence/function
owners (Friston FEP, Kauffman, assembly theory) disclaim the sign; sign owners (biosemiotics,
Pattee, Sebeok) refuse to go below life. We try to build a criterion `C(D, S, dyn)` that fills it,
and we hold it to a fixed adversarial bar: a pre-life, non-self-maintaining system it labels STEER
that is FEP-flat and that Deacon's teleodynamics labels PUSH. If no candidate survives, we concede
(renaming).
